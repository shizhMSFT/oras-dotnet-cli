# What's New in v0.3.0

**ORAS .NET CLI v0.3.0** delivers a major TUI overhaul with Unicode block art, ORAS brand colors, and polished menu navigation—alongside important bug fixes, expanded test coverage, and codebase cleanup.

---

## 🎨 TUI Overhaul

- **Unicode Block Art Banner** — Replaced FigletText ASCII art with hand-crafted Unicode block art using ORAS brand colors (O=#D04485, R=#5EBAB4, A=#FCFCFD, S=#CCF575).
- **Unicode Status Symbols** — Replaced ASCII indicators (`[+]`/`[X]`/`[i]`/`[!]`) with Unicode symbols (✔/✗/ℹ/⚠).
- **Artifacts Sub-menu** — Moved artifact operations (Push, Pull, Copy, Tag, Backup, Restore) into a dedicated sub-menu for a cleaner main dashboard.
- **Cyclic Menu Navigation** — All menus now wrap around: ↓ past the last item returns to the first, ↑ past the first goes to the last.
- **Non-selectable Separators** — Visual `───` separator before Quit/Back options; cursor skips over them.
- **Menu Layout Polish** — Removed extraneous blank lines, aligned option indentation consistently.

## 🐛 Bug Fixes

- **Registry Listing** — Fixed dashboard to show ALL connected registries by aggregating `auths`, `credHelpers`, and `credsStore` from Docker config (previously only showed `auths`).
- **AOT JSON Serialization** — Fixed source-generated JSON for AOT-compatible credential serialization.
- **DI Patterns** — Fixed dependency injection patterns and helper timeout issues.
- **Cancellation Tokens** — Propagated cancellation tokens consistently across all CLI commands.

## ♻️ Code Quality

- **Shared Utilities** — Extracted shared utilities, removed dead code, standardized patterns across the codebase.
- **Refactored TUI Actions** — Restructured artifact action handlers for better maintainability.
- **AOT-Safe Output** — Fixed output formatting for ahead-of-time compilation compatibility.

## 🧪 Testing

- **Critical Unit Tests** — Added unit tests for Tier 1 and Tier 2 targets covering core functionality.

## 🧹 Cleanup

- **Removed NuGet Publishing** — Cleaned up NuGet publishing from the release pipeline (we distribute binaries, not packages).
- **Removed Stale Directory** — Cleaned up unused `src/oras` directory.
